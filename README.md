# 🚑 Tap4Life

**Tap4Life** is an emergency response web application designed to provide immediate help during medical crises. With a single tap on the SOS button, users can instantly alert their emergency contacts, share their live location, and provide critical medical information.

---

## 🌟 Key Features

### 🔐 User Authentication
- Secure user registration and login  
- Personalized dashboard for each user  

---

### 🚨 Emergency SOS System
- Large, easily accessible **red SOS button**
- On pressing the SOS button:
  - 📍 Sends **live location** to emergency contacts  
  - 📞 Initiates **calls to emergency contacts**  
  - 🏥 Attempts to contact **nearby hospitals**  
  - 📩 Sends **SMS alerts**  
  - 🔔 Triggers notifications wherever possible  

> Designed to act as a **first point of contact** during emergencies.

---

### 🩺 Digital Medical Card
Stores important health details:
- Name  
- Age  
- Blood Group  
- Allergies  
- Existing Medical Conditions  

> Helps responders act quickly with accurate medical information.

---

### 📞 Emergency Contacts Management
- Users can add and manage multiple emergency contacts  
- Contacts are alerted instantly during SOS activation  

---

## 💻 Running Locally

Follow these steps to run the project on your system:

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/tap4life.git
cd tap4life
```

## Backend Setup
```
cd backend
pip install -r requirements.txt
python app.py
```

## Frontend Setup
Open a new terminal:
```
cd frontend
npm install
npm run start
```
